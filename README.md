# 🌙 Midnight Study Club

![React](https://img.shields.io/badge/Frontend-React-blue?logo=react)
![Vite](https://img.shields.io/badge/Build-Vite-purple?logo=vite)
![Node.js](https://img.shields.io/badge/Backend-Node.js-green?logo=node.js)
![License](https://img.shields.io/badge/License-MIT-yellow)

A collaborative platform where students **share academic resources** such as notes, assignments, and past papers to survive finals together.

Midnight Study Club makes it easy for students to **upload, discover, and access study material quickly** — especially during late-night study sessions.

---

## 🚀 Live Demo

🌐 **Frontend:** https://midnight-study-club.vercel.app  
⚙️ **Backend API:** https://midnight-study-club-backend.onrender.com

---

## ✨ Features

### 📤 Resource Uploading
- Drag-and-drop uploads
- Multiple file support

### 📚 Resource Discovery
- Browse shared study materials

### 📱 Responsive UI
- Works on desktop and mobile

---

## 🖥️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- Vercel (hosting + analytics)

### Backend
- Node.js
- Express
- Helmet
- Compression
- express-rate-limit

### Storage
- Google Drive API

---

## 🔒 Security Features

- **Helmet** for HTTP security headers  
- **Rate limiting** to prevent spam uploads  
- **Secure CORS configuration** to restrict API access  
- **Environment variables** to protect sensitive data  

---

## 📊 Analytics

The platform uses **Vercel Analytics** to track:

- Visitors  
- Page views  
- Device types  
- Geographic distribution
